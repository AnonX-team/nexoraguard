03 — Capabilities

Kya kya kaam karta hai?

🖥
EDR — Endpoint Detection
Processes, registry startup items, file changes, USB devices, brute force logins, privilege escalation, PowerShell abuse, suspicious process spawns, LOLBin detection, process injection, new accounts, scheduled tasks, services.
🌐
NDR — Network Detection
15 DDoS protocols (TCP SYN, UDP, DNS/NTP/Memcached amplification, HTTP Flood, Slowloris, Carpet Bombing, IoT Botnet). Port scanning, suspicious outbound ports, 20+ ML features (entropy, SYN ratio).
🧠
UEBA — Behavior Analytics
Off-hours login detection, new source IP alerts, credential stuffing, password spray, account enumeration, privilege anomalies, rare process execution. Rolling behavioral baselines.
🔴
Ransomware Detection
Mass file modification (>30 in 60s), Shannon entropy analysis (>7.2 bits), 60+ ransom extension scan, ransom note detection (20+ filenames), shadow copy deletion (vssadmin/bcdedit), LockBit/Conti/BlackCat signatures.
↔
Lateral Movement
RDP from external IPs, new-source RDP logins, SMB spread to 5+ hosts, ADMIN$/C$ share access, WinRM external connections, Pass-the-Hash (NTLM), remote service installation (T1570).
🔍
Vulnerability Scanner
Windows patch gap detection, EOL software registry scan (20+ software), CISA KEV reference (10 active CVEs: Zerologon, PrintNightmare, etc.), weak config checks (RemoteRegistry, WinRM, ADMIN$).
⬡
XDR Incident Correlation
EDR + NDR alerts ek hi incident mein group karta hai — shared IP/user aur 5-min window. Cross-sensor incidents (endpoint + network dono) ko high confidence deta hai. Attacker entity tracking.
⛓
Kill Chain Prediction
12 ATT&CK stages track karta hai. Jo stage abhi active hai usse dekh ke NEXT likely stage predict karta hai. Specific T-numbers batata hai jo attacker likely use karega. Dwell time measure karta hai.
⚡
Automated Playbooks
12 pre-built response scenarios: ransomware → auto-isolate, brute force + login → auto block IP, DDoS → firewall rules, lateral movement → block + alert. Manual recommendations bhi milti hain.
